1勝宏科技 IPO：年內最大 HK IPO，295.6x 超購，4/21 上市 NEW

客觀事實
  • 4/13-16：招股期。4/15 國際簿記提前 1 天截止（需求爆滿）
  • 4/17：定價 HK$209.88/股（頂格定價），集資 HK$175 億（~US$22.4 億）— 2025 年 9 月紫金礦業以來 HK 最大 IPO
  • 公開認購：孖展認購 HK$5,189 億，超額認購 295.6 倍
  • 基石投資者：CPE Rosewood、Janchor Fund、雲鋒資本、Jump Trading、名創優品葉國富等 — 合計 >US$10 億
  • 4/21（周一）上市：全球 AI PCB 收入第一（13.8% 市佔率），Nvidia 主要供應商。2025 年收入 RMB 193 億（+79.8% YoY），淨利 RMB 43.1 億（+273.5%），毛利率 35.2%
  • 國泰君安：4/15 首予「增持」評級
業務思考
295.6x 超購 = 本輪港股 IPO 賺錢效應的最高潮。富途 IPO 認購市佔率 >50%，勝宏科技對佣金+孖展收入貢獻極為直接。

關鍵觀察：近 10 只新股暗盤平均漲 66%、首日均漲 76%。思格新能 +103%、群核科技 +144%、長光辰芯 +75% — 港股 IPO 賺錢效應拉滿，Q1 港股集資 HK$1,099 億（全球第一）。

建議：① 4/21 盯盤勝宏科技首日表現（市場情緒風向標）；② 評估 IPO 孖展利率定價策略 — Tiger 和 uSMART 均推 0% IPO 孖展搶客；③ 華勤技術 4/23 上市（二次上市 ~HK$45.5 億），129x 孖展超購在途。

2AI Agent 軍備賽全面爆發：富途 vs 長橋 vs Tiger 三路並進 延續 #002

客觀事實
  • 4/14：富途牛牛（moomoo SG）上線 Hong Kong 股票期權交易 + 「Moomoo API Skills」品牌 — 定位 Agentic Investing（無代碼量化策略）
  • 4/13：長橋正式宣布 PortAI 更名為 LongbridgeAI，從 LLM chatbox 升級為 AI Agent（思考+捕捉+操作），同獲 SBR AI-Brokerage 獎項
  • 4/14-17：Tiger 平台上線 Earnings Agent，自動生成 Intel/ServiceNow/UnitedHealth 等財報前瞻
  • SBR Tech Awards 2026：Tiger 奪 Fintech-Brokerage（第 3 年連莊）、Longbridge 奪 AI-Brokerage — 直接品牌對撞
業務思考
三家同一周密集發布 AI 產品 = 行業已進入「LLM → Agent」轉型窗口。

定位差異：富途 = API Skills（開發者向）；長橋 = AI Agent（全流程伴隨）；Tiger = Earnings Agent（內容自動化）。富途的 Agent 最開放（API）、長橋最完整（全產品）、Tiger 最垂直（財報）。

建議：追蹤三家 AI 功能的用戶採納率和社區反饋；評估富途牛牛 HK 版是否同步上線 API Skills（目前僅見 SG）。

建議：追蹤要約接納率披露 + 4/30 SFC 節點；觀察耀才 App 更新日誌。

5SEC 廢除 PDT 規則：全美散戶日內交易門檻歸零 NEW

客觀事實
  • 4/14：SEC 正式批准廢除 Pattern Day Trader $25K 最低門檻，改用即時盤中保證金框架
  • Webull：宣布 Day-One 支持 新規 — 股價 4/15 +8.9%~10.2%，call options 量 +78%
  • Robinhood +7.8%、Schwab 確認支持。Tiger/富途牛牛/長橋尚未公布具體響應
  • 華爾街分析師稱「散戶券商重大利好」— 預期 US 市場交易量結構性提升
業務思考
PDT 廢除 = 所有提供 US 市場交易的券商結構性利好（更多日內交易 → 更高佣金/融資收入）。Webull 搶先宣布 = 品牌先發優勢，富途需評估響應節奏。

建議：確認富途牛牛 US 交易服務是否自動適用新規；評估是否需要營銷跟進。

6宏觀：霍爾木茲海峽反覆 + Fed 持觀望 + Trump-Xi 5/14 北京峰會確認 延續 #002

客觀事實

  • 4/19（日）：伊朗反轉攻擊商船 → Brent 回彈 +5.8% 至 US$95.64。停火 4/22 到期
  • Fed：4/28-29 FOMC 預期持穩 3.50-3.75%。J.P.Morgan 預測全年不動，下一步可能是 2027 Q3 加息 25bp
  • Trump-Xi：5/14-15 北京峰會確認。Trump：「potentially Historic」。分析：北京在峰會前積極「整理戰略版圖」（王毅訪朝、習見阿聯酋/西班牙/越南領導人）

業務思考
利率高位 = 富途利息收入基礎穩固。霍爾木茲反覆帶來的油價波動 = 交易量波動 = 佣金收入波動。

關鍵節點：4/22 停火到期、4/28-29 FOMC、5/14-15 Trump-Xi。如果 5 月峰會釋放正面信號 → 港股可能突破 27,000。

建議：融資利率策略維持不變；關注 5/14 峰會前的預期交易。
