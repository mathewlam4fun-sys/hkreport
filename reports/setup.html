用 AI 幫你每兩週自動生成一份競品快報

裝好一次就不用再管。每兩週週五早上，AI 會自動抓資料、寫報告、發到你自己的網址。整個安裝由 AI 助手對話完成，不用寫一行程式碼。

30 分鐘  安裝一次 不用寫程式  AI 對話搞定 每期約 US$0.4  公司金鑰可報銷
👀 先看看做出來會長什麼樣？ 點 最新一期（或頂部 switcher 任何一期）就是你會擁有的網站樣式，只是內容會換成你關心的競品。

1事前準備（10 分鐘）

下面 4 樣東西先備齊，安裝過程逐項貼給 AI 即可：

  1. Mac + AI 助手 Claude Code
    macOS 任何近年版本都可以。沒裝過 Claude Code 的話到 claude.com/claude-code 免費下載（5 分鐘）— 它是 Anthropic 出的桌面 AI 助手，整個安裝過程靠它對話完成。
  2. 公司 AI 中台金鑰
    飛書 → AI 中台 → 個人頭像 → 密鑰管理 → 複製密鑰（user-key-... 開頭）。這把金鑰用來讓服務調用 AI 寫報告。
  3. 兩個免費註冊金鑰
    ① 到 app.tavily.com 註冊，複製 tvly-... 開頭那行（讓 AI 上網搜新聞）。
    ② 到 firecrawl.dev 註冊，複製 fc-... 開頭那行（讓 AI 抓網頁內容）。各 30 秒。
  4. 想好你的網址
    例如 mary-report.surge.sh，可用名字、團隊名、產品線都行。免費、先到先得，AI 會幫你註冊。

2一句話啟動安裝

打開 Claude Code，把下面這行原話貼進對話框，按 Enter — 之後就跟著 AI 對話即可。

貼給 Claude Code 的話
幫我從 https://github.com/mathewlam4fun-sys/hkreport 拉取並按 HANDOFF-CLAUDE.md 安裝到我的 Mac
AI 會主動逐項問你 4 個金鑰、網址、發報時間，你貼上 / 回答即可。所有技術步驟 AI 全包 — 你不用懂程式碼、不用開 terminal、不用做任何手動設定。

3試跑一次看效果

裝完後 AI 會問你「要不要花約 US$0.5 試跑一次驗證？」— 強烈建議跑一次。等 5-10 分鐘，你的網址就會出現你的第一期報告。

之後你完全不用做任何事

每兩週週五早上 09:00（時間可自訂）你的 Mac 會自動跑下一期。Mac 關機 / 睡眠沒關係 — 開機後會自動補跑。

每兩週去網站看當期內容即可。新內容上線時不會打擾你（沒有彈窗、沒有通知）。
🎨 想改成自己關心的競品？ 裝完後跟 Claude Code 說「我想把競品名單換成 XXX、YYY、ZZZ」，它會幫你改設定。報告口徑、標題、欄位都可以這樣請它調整。

?常見問題

我完全不會寫程式碼，真的能用嗎？
完全不用懂。整個安裝過程是「AI 問你問題、你貼答案」這樣的對話 — 跟用 ChatGPT 沒兩樣。如果中間遇到任何看不懂的，直接問 Claude Code「這是什麼意思」，它會用人話解釋。出錯了也可以叫它幫你修。
我的金鑰會不會被別人看到？
不會。所有金鑰存在你 Mac 本機的隱藏設定檔，只有你的帳號讀得到。不上網、不會放到 GitHub 程式碼倉庫、不會傳到任何雲端。公司 AI 中台金鑰也符合「不出機器」的合規要求。
每期會花多少錢？誰付？
每期約 US$0.3–0.5（兩次 AI 呼叫），每月 1–2 期合計 ~US$1。公司 AI 中台金鑰一般可走報銷，等於零成本。Tavily 和 Firecrawl 免費額度也夠用。
Mac 關機 / 睡眠的時候會錯過嗎？
不會。Mac 開機後會自動檢查並補跑（仍會按兩週節奏判斷）。你只要保證每兩週有開過一次機就好 — 一般正常用 Mac 工作的人都符合。
裝完之後我要做什麼？
什麼都不用做。每兩週去你的網址看當期內容即可。如果某期內容有問題，告訴 Claude Code，它會幫你查日誌、找原因、修好。
不想跑了 / 想暫停一陣子怎麼辦？
直接跟 Claude Code 說「幫我暫停 hkreport 自動發報」或「幫我卸載 hkreport」，它會處理。你不用記命令、不用開 terminal。
跟現有的 hkreport.surge.sh 是同一個嗎？
是同一套程式碼，但裝完跑出來的是「你自己的站」（你的網址、你關心的競品、你的口徑），跟現在這個站完全獨立、互不影響。原作者的也會繼續更新。
