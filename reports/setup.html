Self-Host Guide

想為你的團隊跑一份這樣的競品快報？

整套服務已封裝為 GitHub repo，全程由 Claude Code 引導安裝。預備 5 分鐘 + 安裝 30 分鐘，你會擁有自己的 *.surge.sh 站點，每雙週自動更新。

~30 分鐘  安裝 US$0.3–0.5  每期 API 成本 密鑰不出機器  本機 launchd

1事前準備（5 分鐘）

下列 4 樣東西安裝過程會用到，建議先準備齊：

  1. 公司 AI 平台 user-key
    找 IT / AI 平台組要一條（user-key-... 開頭）。本服務跑在你自己 Mac 上、走公司代理，符合 key 不出機器的政策。
  2. Tavily API key（新聞 / 社媒搜尋）
    到 app.tavily.com 免費註冊，複製 tvly-... 開頭的 key。每月 1000 次查詢，足夠用。
  3. Firecrawl API key（網頁 JS 渲染抓取）
    到 firecrawl.dev 免費註冊，複製 fc-... 開頭的 key。
  4. 想好你的 Surge 域名
    例如 yourname-report.surge.sh，任何 *.surge.sh 都可，先到先得。Surge 帳號安裝過程會引導你註冊。

2一句話安裝

打開 Claude Code 對話框，把下面這行原話貼進去：

Claude 會主動逐項問你密鑰、域名、排期，跟著答即可。預檢、寫 env、建 venv、註冊 launchd、kickstart 驗證全自動完成。

?常見問題

密鑰安全嗎？會不會洩漏？
所有密鑰寫到 ~/.config/hkreport/env（chmod 600），只有你的 Mac 用戶讀得到；不入 git、不上 CI、不離開本機。Surge token 同理。Repo 本身為 public，但只含程式碼與模板，沒有任何真實密鑰。
Mac 關機/睡眠的話會錯過嗎？
launchd 在 Mac 開機後會檢查錯過的觸發並補跑，仍受雙週閘門控制（不是錨點當天則 SKIP）。所以你只要保證錨點當天有開過機，就會跑。
每期到底會花多少錢？
兩次 Claude Sonnet 4.6 呼叫（classify + compose），約 US$0.30–0.50/期。若觸發 VERIFY 失敗的 fix-up 重試，最多再多 US$0.10–0.20。每月 1–2 期，每月成本約 US$1。
沒有 Mac 也能用嗎？
目前 launchd 排期只支援 macOS。理論上把 run.sh 改用 cron + 把 plist 換成 systemd 也能在 Linux 跑，但 repo 沒附這份適配。
想換時間 / 換域名 / 暫停 / 卸載？
換時間：重跑 bash install.sh，向導會覆寫 env 與 plist。
換域名：編輯 ~/.config/hkreport/env 改 SURGE_DOMAIN。
暫停：launchctl bootout gui/$(id -u) ~/Library/LaunchAgents/com.<你>.hkreport.plist。
卸載：repo 根目錄跑 bash uninstall.sh，可逐項選刪。
跟現有的 hkreport.surge.sh 是同一份嗎？
是同一套程式碼，但裝完跑出來的是你自己的站點（你的域名、你的競品名單、你的視角）。原作者的 hkreport.surge.sh 繼續獨立運行，互不影響。
