1LIHKG「轉會潮」升溫 + 隱藏收費投訴集中爆發（5/09）

客觀事實
  • 5/09：LIHKG 新帖「唔用富途轉邊間?」引發大量討論，Webull、IB 被推薦為替代方案 [LIHKG]
  • 4/21：「富途愈黎愈多隱藏收費」帖——用戶指 SEC 規費（4/4 起調至 0.00206%）被感知為隱藏加價 [LIHKG]
  • 4/19：「買賣美股，富X 牛手續費好貴」帖——用戶對比 IB 每筆省數百港元 [LIHKG]
  • 同期：Tiger 推 HK$3,200 迎新、Webull 推 HK$2,200 現金獎賞，形成「推拉」夾擊 [HongKongCard] [MoneySmart]
業務思考
vs #004 升級：上期「Webull 真零費永久定價」已作為 YELLOW 信號，本期 LIHKG 用戶開始實際行動討論轉會，信號從觀望升級為行動意圖。

費用投訴集中在三個痛點：① SEC 規費透明度不足（用戶感知為「偷加」）；② 美股平台費 vs IB 對比劣勢明顯；③ 競品同步推出大額現金迎新形成心理落差。

建議：① 費率頁增加 SEC 規費說明（監管轉嫁 vs 平台收費區分）；② 評估美股高頻交易客群的留存數據是否已出現異動；③ 考慮針對性回防方案（如美股交易免佣券、loyalty 計劃）。

2USM 無紙證券制度 11/16 強制上線 — 最後 6 個月倒數（4/16–4/24）延續 #001

客觀事實
  • 4/16–4/24：HKEX 完成三場 USM 實施簡報會（Teams + 實體，分英/粵語場次）[HKEX Circular]
  • 4月：HKSCC 透過 Client Connect 向所有參與者發出「USM 準備度問卷」[HKEX USM]
  • 3/30：SFC 確認目標上線日為 2026 年 11 月 16 日，新上市證券強制以無紙形式發行 [Timothy Loh]
業務思考
vs #004 維持紅色：距上線不足 180 天，HKEX 已進入「確認準備度」階段。若系統改造延誤，11/16 後將無法正常處理新股交割。

關鍵時間節點：① 問卷結果反饋（預計 5-6 月）→ 確認我司是否被標記為「未達標」；② 第二次全市場演練（預計 7-8 月）→ 最後壓測機會；③ 11/16 強制切換 → 零容錯。

建議：① 確認技術團隊已完成問卷提交；② 內部設定 8 月底為「USM Ready」硬截止日；③ 評估客戶端（App）是否需要 UI 變更以適配無紙證券持倉展示。

本期需關注

3Moomoo API Skills「Agentic Investing」全球首發（4/23）

客觀事實
  • 4/23：moomoo 發布 API Skills，允許個人 AI Agent 直連交易基礎設施，支持自然語言→策略邏輯→自動執行 [StockTitan]
  • 4/23：功能覆蓋美、加、港、新、日五個市場，支持回測+模擬盤，credentials 保持本地 [Finovate]
  • 4/27：CNBC Fast Money 專訪 moomoo CEO Neil McDonald，FUTU 當日跌 4.12% [CNBC]
業務思考
這是我司在海外市場的重大產品動作，需觀察幾個維度：
① 採用率：API Skills 門檻較高（需使用者有 AI agent 開發能力），初期可能僅覆蓋 quant/tech-savvy 群體；
② 監管反應：SFC 對 AI agent 自動交易的態度尚不明確，港股市場是否同步開放需確認；
③ 競品跟進：Tiger 已有 AI 選股助手，若快速模仿則先發優勢窗口有限。

需確認：香港站（富途牛牛）是否同步上線 API Skills？若未上線，競品可能利用此功能差異做文章。

4Longbridge 品牌重組 + 東南亞全面擴張（5/05）

客觀事實
  • 5/05：LONGPORT Whale（原 Longbridge Whale，1 月改名）在 Bursa Malaysia Stockbroking Trade Fair 宣布進入馬來西亞市場 [MaherSaham]
  • 5月：Longbridge 任命 Gavin Chia 為新加坡及東南亞區域 CEO [PRNewswire]
  • 持續：LONGPORT Whale 已服務亞洲 100+ 金融機構，定位「AI-ready 證券交易基礎設施」
  • 5月：Longbridge SG 推出 10% p.a. 利息加碼（現金管理）[GrowBeansprout]
業務思考
Longbridge 正從「券商 App」轉型為「券商基礎設施供應商」（B2B2C），這與我司 moomoo 的海外擴張形成差異化競爭路徑。

影響評估：① 短期對港股業務無直接威脅；② 中期若 LONGPORT Whale 為馬來/泰等市場券商提供白標解決方案，可能間接壓縮 moomoo 在新興市場的獨立拓展空間；③ 新 CEO 任命信號強烈 — 東南亞是長橋戰略重心。

建議：關注 LONGPORT Whale 客戶清單公開進展，評估是否有我司目標合作方重疊。

5Tiger 全方位獲客戰 + AI 升級（持續至 6/30）

客觀事實
  • 5月：Tiger 透過 HongKongCard、MoneySmart、蛋黃視野等渠道推送最高 HK$3,200 迎新獎賞（含開戶即拎 $550 + 留存 $1,200 + 交易 $900 + Crypto $500）[HongKongCard] [蛋黃視野]
  • 2026：Tiger 全面升級 24/7 AI 投資助手 — 對話式選股+百頁財報摘要+條件篩選 [Vocus]
  • 5/08：Tiger Brokers HK Facebook 母親節行銷保持高活躍度 [Facebook]
業務思考
Tiger 的獲客策略「三管齊下」：金錢激勵（$3,200）+ 產品差異化（AI 助手）+ 渠道密度（多個 KOL/比較平台同時推送）。配合 RED#1 的富途費用投訴，形成精確的「推拉」效應。

風險判斷：Tiger Q1 創歷史新高（#004 已報），有充足預算維持高補貼至 6/30。

建議：① 監測 5 月新開戶轉化漏斗是否出現 Tiger 分流跡象；② AI 投資助手對標分析 — 我司 moomoo AI Skills 能力更強但門檻更高，是否需要一個低門檻「對話式」入口。

6Webull 限時激進快閃定價（5/14–5/21）

客觀事實
  • 5/14–5/21：Webull 透過 MoneySmart 推出「8 天限定」HK$2,200 現金迎新獎賞 [MoneySmart]
  • 持續：Webull 維持「真零費」永久定價（#004 已確認），疊加現金獎賞形成短期脈衝式吸引力
業務思考
Webull 策略從「長期零費」轉為「零費 + 限時現金脈衝」，精確打擊價格敏感型用戶。8 天窗口製造緊迫感，配合 LIHKG 討論中被推薦為替代方案，轉化率可能高於常規推廣。

建議：觀察 5/21 後 Webull 是否延續或加碼類似活動，判斷是否為持續性策略。

7螞蟻/耀才收購完成 — 整合啟動期延續 #001

客觀事實
  • 3/17：螞蟻集團子公司完成最終監管程序，正式取得耀才 50.55% 股權（HK$28.1 億）[CNBC]
  • 3/16：Bloomberg 報導螞蟻獲中國監管核准 [Bloomberg]
  • 3月：耀才股價單日暴漲 58%，收購完成確認 [Yahoo Finance]
  • 本期：整合期進行中，尚無公開可見的產品或品牌整合動作
業務思考
vs #004 維持：收購已於 3 月完成，但目前處於「靜默整合期」— 尚未看到 App 改版、品牌更名或 AlipayHK 金融入口上線。

判斷：螞蟻的整合通常耗時 6-12 個月（參考天弘/網商模式）。預計 H2 2026 會有可見動作。屆時若 AlipayHK（1,500 萬用戶）開放證券入口，將是對整個互聯網券商板塊的結構性衝擊。

建議：維持監控，重點關注：① 耀才 App 改版/更名動態；② AlipayHK App 是否出現「投資」tab；③ 耀才招聘方向（產品/技術崗增量 = 整合加速信號）。
